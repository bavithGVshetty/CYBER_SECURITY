## Vulnerability attacks
1. SQL Injection(SQLi):
- When user input is not validated, attackers inject SQL commands into the query.
2. XSS(Cross site Scripting )
- It can steal data or tokens
- Hijack User Accounts.
- Can change Design and content.
- Perform actions as the user.

3. Missing Security Headers
- Should always send HTTPS apart from HTTP.
- The HTTP transmit data in plain txt.
