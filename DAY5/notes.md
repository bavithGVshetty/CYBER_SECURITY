# SIEM TOOLS
An SIEM stands for Security Informaton and Event Management.

This tool is used in area's such as.
- Servers
- Database
- Cloud Services
- Endpoints
- Firewalls
Why it is used ?
-> It is used to analyze Suspicious activity.

## Tools Famous:
- Splunk
- QRader
- Microsoft Sentinel

## Operations of  SIEM.
SIEM gathers logs from.
- login attemps
- File Changes
- Network traffic
- Error in the app.
- Real time Alerts for the app.
- Detect the suspicous Activities
- It takes the operational input in terms of log file.
